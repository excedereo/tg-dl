# TG-DL

Telegram-бот скачивания видео с **YouTube** и **TikTok** (без вотермарки).
Модуль для [vaeli-hub](https://github.com/excedereo/vaeli-hub) — фоновый сервис,
hub запускает его как задачу вместе с остальными.

## Как пользоваться

В любом чате: `@имя_бота <ссылка>` → плашка «Скачать видео» → тап → в чат
вставляется заглушка «качаю...», затем бот подменяет её на готовое видео.

## Конфиг

Секреты — в переменных окружения панели **или** в файле `.env` (не коммитится):

```
TG_TOKEN=12345:ABC...          # токен от @BotFather
TG_STORAGE_CHAT=-100xxxxxxxxxx # id канала-склада, бот там админ
```

Боту в @BotFather включить inline: `/setinline`.

## Зачем канал-склад

Чтобы вставить видео в чужой чат через inline, Telegram требует, чтобы файл уже
был загружен (есть `file_id`). Бот заливает видео в канал-склад, берёт оттуда
`file_id` и им вставляет ролик в любой чат.

## Ограничения

- видео ≤ 720p и ≤ 10 мин — чтобы влезть в лимит размера бота (50 МБ)
- TikTok зависит от доступности с серверного IP
